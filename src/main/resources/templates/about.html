About Online Bookstore

Welcome to our Online Bookstore - a full-featured e-commerce application demonstrating modern DevOps practices and cloud-native architecture!

üìö Features

This is a complete bookstore web application with the following features:

  • Browse books by category (Technology, Fiction, Science, Business, Biography)
  • Search books by title or author
  • View detailed book information
  • Add books to shopping cart
  • Place orders with customer information
  • Fully responsive design

üöÄ CI/CD Pipeline

This application is deployed using a complete CI/CD pipeline:

  • Source Control: GitHub for version control
  • CI/CD: Jenkins for automated builds and deployments
  • Containerization: Docker for packaging the application
  • Orchestration: Kubernetes (AWS EKS) for container management
  • Infrastructure: Terraform for infrastructure as code
  • Configuration: Ansible for server provisioning
